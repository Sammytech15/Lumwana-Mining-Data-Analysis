# Lumwana-Mining-Data-Analysis

### Lumwana Copper Mine — 2024 Operations Analysis

## Project Overview
A full year data analysis of Lumwana copper mine operations 
covering 730 shifts across four deposits in North-Western Zambia.

## Tools Used
- Microsoft Excel 2016
- PivotTables and PivotCharts
- Conditional Formatting
- Statistical Functions
- IF Logic and Data Cleaning

## Key Findings
- Total ore mined: 8.72 million tonnes
- Copper concentrate produced: 497,410 tonnes
- Peak production month: October (dry season)
- Only 19% of shifts achieved High performance status
- Wet season months drove 68% of Low performance shifts
- 21 safety incidents recorded across 730 shifts

## Dashboard Features
- Monthly production trend line chart
- Deposit grade vs tonnage combo chart
- Shift performance distribution donut chart
- Ore grade distribution histogram
- Safety incidents and downtime tracker
- 8 key metrics summary box

## Dataset
Simulated dataset modelled on real Lumwana mining parameters
including ore grades, tonnage, equipment data, safety records
and operational metrics across 4 deposits:
- Chimiwungo
- Malundwe
- Kamisengo
- Kababisa

## About
This project was completed as part of a structured 30-day 
data analysis learning program focused on mining industry data.
